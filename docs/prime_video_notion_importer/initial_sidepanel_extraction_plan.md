# 初回サイドパネル抽出改善 実行計画

作成日: 2026-07-01

## 目的

Prime Video の作品ページでサイドパネルを初回起動したとき、更新ボタンを押さなくてもタイトル、サムネイル、重複チェックが揃うようにする。

## 手順

1. `AGENTS.md`、拡張機能の構成、サイドパネル初期化処理、更新ボタン処理を確認する。
2. 初回起動時と更新ボタン押下時の抽出タイミング差分を特定する。
3. 初回起動時にも DOM と埋め込みデータの準備を待ってから抽出する処理を追加する。
4. タイトルまたは画像が空の場合は短時間リトライし、取得できた時点でフォーム反映と重複チェックを実行する。
5. プロジェクト運用ルールに従い、`manifest.json`、`popup.js`、`popup.html`、`CHANGELOG.md` のバージョンを揃える。
6. 構文確認と差分確認を行い、実ブラウザで未確認の範囲があれば明記する。
