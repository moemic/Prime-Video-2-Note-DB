# iPhoneショートカット実現可否調査計画

作成日: 2026-07-07

## 目的

現在のPrime Video to Notion Chrome拡張機能と同等または近い機能を、iPhoneショートカットで実現できるかを、現行実装に基づいて判定する。

## 実行手順

1. 現行Chrome拡張の構成を確認する。
   - `manifest.json`
   - `content.js`
   - `popup.js`
   - `background.js`
   - `popup.html`
   - `README.md`

2. 機能を分解する。
   - Prime Videoページから取得している情報
   - Chrome拡張APIに依存している処理
   - Notion連携に必要な設定と通信
   - 画面UI、重複確認、候補選択、画像取得などの補助機能

3. iPhoneショートカットで代替可能かを分類する。
   - ショートカット単体で可能
   - ショートカットと外部APIの組み合わせなら可能
   - iOS Safari拡張または専用アプリが必要
   - 実現困難または不安定

4. 実現案を比較する。
   - 純ショートカット版
   - ショートカット + Web API版
   - iOS Safari拡張版

5. 判定結果を日本語でまとめる。
   - 完全再現できる範囲
   - 代替実装で妥協できる範囲
   - 実用化する場合の推奨構成
   - 次に作るべき成果物

## 注意事項

- この調査ではコード修正やバージョン更新は行わない。
- 調査結果は現行リポジトリの実装に基づく。
- Appleショートカット側の仕様は変わる可能性があるため、最終実装前に実機検証が必要。
