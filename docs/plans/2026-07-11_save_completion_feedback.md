# Notion保存完了フィードバック改善計画

## 目的

Notionへの新規保存・更新が完了した瞬間を、文字だけでなく色、動き、チェックマーク、波紋で明確に伝える。

## 実装内容

1. 保存成功時に保存ボタンを完了状態へ切り替える。
2. 新規保存は「Notionに保存しました！」、更新は「変更を保存しました！」と表示する。
3. チェックマーク、光の波紋、粒子、軽い跳ねを組み合わせた短い演出を追加する。
4. 約1.6秒後に通常の保存ボタンへ戻し、連続保存を妨げないようにする。
5. `prefers-reduced-motion` が有効な環境では大きな動きを停止する。
6. エラー表示と送信中表示の既存挙動は維持する。
7. UI改善としてバージョンを `1.31.0` に更新し、変更履歴を記録する。

## 検証

- `node --check popup.js`
- `npm run check:version`
- 新規保存・既存ページ更新・失敗時の表示経路をコード上で確認する。
