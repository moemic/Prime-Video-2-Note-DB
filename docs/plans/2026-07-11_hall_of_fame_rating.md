# 殿堂入りチェック実装計画

## 目的

星5評価とは独立して、本当におすすめしたい作品を「殿堂入り」として記録・更新できるようにする。

## 実装内容

1. レーティング欄へ「殿堂入り」チェックボックスを追加する。
2. Notion DBに `殿堂入り` プロパティがなければcheckbox型で自動作成する。
3. 同名プロパティが別の型で存在する場合は、明確なエラーを表示する。
4. 新規登録と既存ページ更新でチェック状態を保存する。
5. 既存ページ読込時に現在のチェック状態を復元する。
6. 保存完了通知へ「殿堂入りに設定しました」「殿堂入りを解除しました」を表示する。
7. バージョンを `1.35.0` に更新する。

## 検証

- `node --check background.js`
- `node --check popup.js`
- 既存の回帰チェック一式
- `npm run check:version`
- `git diff --check`
