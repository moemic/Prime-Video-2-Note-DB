# 画像拡大プレビュー実装計画

## 目的

画像候補をダブルクリックで拡大し、細部を確認してからプロパティ画像やページカバーを選択できるようにする。

## 実装内容

1. ダブルクリックで暗い背景の拡大プレビューを開く。
2. 背景クリック、閉じるボタン、Escキーで閉じる。
3. 左右ボタンと左右キーで前後の画像へ移動する。
4. 現在位置を「2 / 6」の形式で表示する。
5. シングルクリックとダブルクリックを判別し、拡大時の誤選択を防ぐ。
6. 右クリックによるページカバー選択は維持する。
7. ポップアップとサイドパネルの幅に合わせて画像を表示する。
8. バージョンを `1.34.0` に更新する。

## 検証

- `node --check popup.js`
- 既存の画像完全入れ替えチェック
- `npm run check:version`
- `git diff --check`
