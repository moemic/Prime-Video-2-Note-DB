# Notionテンプレート配布とREADME更新 計画

## 目的

Notionデータベースをユーザーが事前に作成する方式から、複製可能なNotionページをデータベーステンプレートとして配布する方式へ案内を更新する。あわせて、現行コードの`カテゴリ`仕様にREADMEを同期する。

## 実施内容

1. 既存の未コミット変更を確認し、READMEとバージョン関連ファイルに競合がないことを確認する。
2. READMEにテンプレート配布・複製・インテグレーション接続・Database ID取得の手順を記載する。
3. READMEのプロパティ定義を現行実装に合わせ、`ジャンル`表記を`カテゴリ`へ統一する。
4. ドキュメント変更に伴いパッチバージョンを更新し、CHANGELOGへ記録する。
5. `npm run check:version`でバージョン整合性を確認する。

## モデル選定

READMEとバージョン同期の編集・検証にはCodex（Terra相当）を使用する。

## 承認

2026-07-12にユーザーから「ok」で承認済み。
