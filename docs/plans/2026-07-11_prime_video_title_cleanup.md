# Prime Videoタイトル末尾表記の除去計画

## 目的

Prime Videoの仕様変更で作品タイトルに付加される視聴案内を除去し、Notionには作品名とシーズン情報だけを保存する。

## 対応例

- `エクスパンス ～巨獣めざめる～のシーズン6を視聴 - Prime Video` → `エクスパンス ～巨獣めざめる～ シーズン6`
- `プロジェクト・ヘイル・メアリーをオンラインで視聴 - Prime Video` → `プロジェクト・ヘイル・メアリー`

## 実装内容

1. 末尾の「を視聴」「をオンラインで視聴」「を観る」とPrime Video表記を除去する。
2. Prime Video直前の区切り文字 `-`、`–`、`—`、`|` の表記揺れに対応する。
3. 「のシーズン＋数字」が末尾にある場合だけ「の」を空白へ置換する。
4. 表示タイトルと重複照合用タイトルに同じクリーニングを適用する。
5. 実際のタイトル処理関数を読み込む回帰チェックを追加する。
6. バージョンを `1.31.1` に更新し、変更履歴を記録する。

## 検証

- `npm run check:title-cleanup`
- `node --check content.js`
- `npm run check:version`
- `git diff --check`
