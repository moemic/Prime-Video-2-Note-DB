# タイトル内の配信元タグ除去計画

## 目的

Prime Videoから取得したタイトルに混ざる配信元表記を除去し、作品名とシーズン情報だけをNotionへ保存する。

## 対象

- `【TBSオンデマンド】`
- `(dアニストア)`
- `（dアニストア）`

## 実装内容

1. 指定された配信元名を既知タグとして管理する。
2. 全角・半角の丸括弧、隅付き括弧に対応する。
3. タイトルの先頭・途中・末尾のいずれにあっても除去する。
4. 配信元名以外の括弧表記は維持する。
5. 除去後の余分な空白を整える。
6. 表示タイトルと重複照合用タイトルへ反映する。
7. バージョンを `1.34.2` に更新する。

## 検証

- `npm run check:title-cleanup`
- `node --check content.js`
- `npm run check:version`
- `git diff --check`
