# 保存完了時の変更内容表示計画

## 目的

保存成功の演出内に、今回登録・変更した内容を短く表示し、何がNotionへ反映されたかを確認できるようにする。

## 実装内容

1. 既存ページを読み込んだ時点のタイトル、概要、監督、タグ、ステータス、評価を保持する。
2. 保存直前の値との差分から変更項目を生成する。
3. 画像完全入れ替え、コメント追加、各プロパティ変更を完了演出へ表示する。
4. 新規登録では、登録した主な内容を表示する。
5. 表示は最大3件とし、残りは「ほかN件」にまとめる。
6. 読めるように表示時間を3.2秒へ延長する。
7. バージョンを `1.33.0` に更新する。

## 検証

- `node --check popup.js`
- `npm run check:version`
- `git diff --check`
