# カテゴリ手動入力後のクリア

## 目的

カテゴリ入力欄で Enter を押してカテゴリを追加した直後、入力中の文字列が残らないようにする。これにより、同じカテゴリを重ねて登録しそうになる操作上の混乱を防ぐ。

## 実施内容

1. Enter 入力時に入力値を確定し、タグ描画より先に入力欄を空にする。
2. カテゴリ追加後も入力欄へフォーカスを維持し、連続入力を妨げない。
3. パッチバージョンを更新し、変更履歴へ記録する。
4. 構文確認とバージョン同期チェックを実施する。

## 完了条件

- Enter でカテゴリが一度だけ追加される。
- Enter 後のカテゴリ入力欄が空になる。
- バージョン表記が全対象ファイルで一致する。
